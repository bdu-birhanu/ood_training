What is OOD?

  • OOD is a user-friendly web interface for managing and interacting with HPC resources.
  • Visit: https://rc.uab.edu/pun/sys/dashboard
  • Key Features
    • Interactive Applications: Run graphical applications and use computing resources through the browser.
    • Job Management: Submit, monitor, and manage HPC jobs.
    • File Management: Access and manage files on remote systems.
  • Why use Open OnDemand?
    • Ease of Use: Simplifies complex HPC tasks with a user-friendly interface.
    • Accessibility: Access HPC resources from anywhere with a web browser.

Connect to Open OnDemand?

  • To connect, visit https://rc.uab.edu/, authenticate with Single Sign-On (SSO).

Open OnDemand Dashboard

Files: allow one to view and operate on files available on the cheaha.

Jobs: allow to create, edit, submit, and monitor jobs on cheaha cluster.

Clusters: provides access to the shell on login nodes.

Interactive Apps: allow to explore interactive apps on Cheaha and lunch job using the preferred app.

Open OnDemand Dashboard: Interactive Apps

Launching Interactive Sessions: HPC Desktop

  • Select an HPC Desktop interactive app, complete the form fields, and click the blue Launch button.

Launching Interactive Sessions: HPC Desktop

  • Interactive session will be initiated and passes through: Queued, Starting and Running

      • Queued: When the blue Launch button clicked the job state will change to Queued

      Starting: After the resources are allocated, the job state will change to Starting.

      Running: Once the job is launched on the compute nodes, the state will switch to Running.
